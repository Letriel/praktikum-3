Pengertian Semantic & Non-semantic Elements

Semantic elements
Semantic elements adalah elemen HTML yang memiliki arti atau makna yang jelas bagi manusia maupun browser.
Dengan hanya melihat tag HTML-nya saja, kita bisa mengetahui fungsi elemen tersebu dalam sebuah halaman web.
Non-semantic elements
Non-semantic elements adalah elemen HTMl yang tidak memiliki makna khusus dah hanya berfungsi sebagai wadah untuk elemen lain.
Browser tidak dapat memahami tujuan dari elemen ini tanpa informasi tambahan seperti class atau id.
